# Easy Doctors Landing Pages

## Descrição

Projeto com 3 landing pages da Easy Doctors seguindo fielmente o copy atualizado do documento fornecido. Cada página possui uma headline diferente para testes A/B, mantendo o mesmo conteúdo nas demais seções.

## Especificações Técnicas

### Design System
- **Cores Principais:**
  - Primária: #74237F (roxo)
  - Secundária: #8a49a1 (roxo claro)
  - Sucesso: #00BE91 (verde)
- **Fonte:** Inter (Google Fonts)
- **Estilo:** Moderno, limpo, gradientes sutis
- **Bordas:** Arredondadas (8px-12px)
- **Sombras:** Suaves e elegantes
- **Responsivo:** Mobile-first

### Stack Tecnológica
- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e responsividade
- **JavaScript** - Interatividade e funcionalidades
- **Bootstrap 5.3** - Framework CSS para layout responsivo
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia Inter

## Landing Pages

### 1. headline1.html
**Headline:** "Quantas vezes você já adiou a ida ao médico por medo da conta no final?"
- Foca na procrastinação por medo dos custos
- Apela para a identificação emocional

### 2. headline2.html
**Headline:** "Nunca cometa o erro de ir a um médico nos EUA se você for brasileiro — aqui está o motivo."
- Alerta sobre erro específico que brasileiros cometem
- Cria curiosidade sobre o motivo

### 3. headline3.html
**Headline:** "Pare agora de pedir remédios para brasileiros... Precisa de receitas médicas?"
- Foca na solução de receitas médicas
- Apela para independência de favores

## Estrutura das 7 Dobras

### 1ª Dobra - Hero (SEM BOTÕES)
- Prova social no topo (15.000 brasileiros protegidos)
- Headline específica de cada página
- Subtítulo padrão sobre 1 clique
- Elementos visuais: 24h, sem burocracia, em português
- Imagem com caption emocional

### 2ª Dobra - Expansão da Promessa
- "Aqui você não pode ficar doente"
- Custos de 10-50 mil dólares
- "Bomba-relógio" emocional
- Dados Johns Hopkins/Harvard (96%)
- 3 relatos em caixas destacadas

### 3ª Dobra - Sistema de Blindagem 4D™
- 4 camadas de proteção detalhadas
- Benefícios específicos
- Lista de vantagens com ícones

### 4ª Dobra - Provas Sociais
- 60 mil brasileiros usando
- Depoimentos da Camila e Paulo
- Como funciona em 3 passos

### 5ª Dobra - Oferta
- EasyDoctors App
- Plano Família (até 3 dependentes)
- Livro Negro dos Seguros
- Preço: US$ 49/mês
- CTA: "QUERO BLINDAR MINHA SAÚDE AGORA"

### 6ª Dobra - Objeções
- Cenário: filho com febre às 2h
- Urgent Care custando $500-800
- "Terror silencioso"
- Solução EasyDoctors

### 7ª Dobra - Credibilidade
- Números de impacto (60k vidas, 20k consultas, 148 médicos, 5 anos)
- FAQ + CTA Final
- Sobre Nós da empresa

## Como Usar

### Desenvolvimento Local
1. Extrair o projeto em uma pasta
2. Abrir qualquer arquivo HTML em um navegador
3. Não requer servidor local (funciona com file://)

### Para Hospedagem
1. Fazer upload de todos os arquivos para o servidor
2. Configurar o domínio para apontar para a pasta
3. Testar todas as 3 páginas

### Testes A/B
- Use cada headline para públicos diferentes
- Monitore conversões de cada versão
- Otimize com base nos resultados
